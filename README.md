# Web Tools

A collection of fast and practical web tools designed to solve everyday problems online.  
All tools are lightweight, easy to use, and work directly in your browser.

🌐 Live website:  
https://juanferquinterl.github.io/web-tools/

---

## Tools Overview

### SplitBill
Easily split bills among multiple people with optional tip.  
**Key Features:**  
- Split total bill by number of people  
- Add optional tip percentage  
- Instant calculation  

Open tool: `/splitbill/`

### QuickBarcode
Generate barcodes in multiple formats and download as PNG or PDF.  
**Key Features:**  
- Supports EAN-13, Code128, and Code39  
- Download as PNG or PDF  
- Display value below barcode  

Open tool: `/quickbarcode/`

### UnitConverter
Convert units of length, weight, temperature, and volume.  
**Key Features:**  
- Length, weight, temperature, and volume conversions  
- Instant results  
- Supports multiple languages (ES/EN)  

Open tool: `/unitconverter/`

### Color Palette Generator
Create beautiful color palettes from a base color.  
**Key Features:**  
- Complementary, Analogous, Triadic, Tetradic schemes  
- Copy HEX codes easily  
- Download palette as PNG or PDF  

Open tool: `/colorpalette/`

### QuickList
Generate bullet or numbered lists quickly.  
**Key Features:**  
- Custom list title  
- Copy to clipboard  
- Download as PDF  

Open tool: `/quicklist/`

### Money-Plan
Calculate monthly and annual savings based on your income and expenses.  
**Key Features:**  
- Set saving percentage  
- Dynamic monthly and yearly savings calculation  
- Copy results to clipboard  

Open tool: `/moneyplan/`
